# Лабораторные работы

Нужно сделать 4 лабораторных работы: первые 3 находятся в директориях lab1 - lab3. Последняя, описана далее.  

## Задание

В лабораторной работе 2 сделан скрипт, подменяющий ключи в файле, напишите Dockerfile, который подменит в config.json {port} на строку :8080, {secret} на произвольную строку.

В лабораторной работе 3 сделана программа на си, напишите Dockerfile, который скомпилирует и запустит программу, передав в stdin "cpu" и запишет ответ в файл output.txt.

В папке backend лежит готовый Dockerfile с сервером, сервер ожидает найти config.json с подставленными значениями в /config/config.json и вывод в /build/output.txt, напишите docker compose файл, который объединит сделанные контейнеры и передаст запросы с порта 80 на порт сервера (port в config.json).
